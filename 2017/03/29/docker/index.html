服务挂载

服务在Docker里就是Container的概念，一个服务就是一个Container，Container镜像来源有两种：Docker Hub,自己build。

  1. 从Docker Hub安装

    使用docker pull [image]拉取到本地，再使用docker run跑起服务。

  2. docker build

    创建一个名为Dockerfile的文件，设置好配置项，使用docker build可以生成镜像，生成的镜像可以通过docker images查看。
