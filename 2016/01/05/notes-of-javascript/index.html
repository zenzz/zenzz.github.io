学习JavaScript的一些笔记

发表于 2016-01-05   |     |   阅读次数

基于ECMAScript 6标准下的学习

JavaScript有两种运行模式，即正常模式和严格模式，严格模式需要使使用’use strict’。

基本知识

  1. 声明的三种方式：var, let(声明块范围局部变量), const(声明只读变量)。let只能在严格模式下运行成功，使用let不存在声明提升。正常模式下对const变量赋值不报错，但变量不会改变，而严格模式下会报错。
  2. 空值null在数值类型环境中会被当0。
  3. 声明提升，变量或函数可以在声明之前使用，运行时，如果此变量未声明，会在此前进行声明，函数会在上下文查找声明，若是未找到则报错。
  4. JavaScript的7种基本类型：Bolean, Null, Undefined, Number, String, Object, Symbol。

  5. 单目加法运算符：将字符串转换为数字的另一种方法。其它的还有parseInt(), parseFloat()，这两个是全局对象window的方法。

  6. 对象可以使用数字或字符串字面值作为属性的名字。

  7. 如果你需要在条件表达式中使用赋值，一个共同的习惯是在赋值语句前后额外的添加一对括号。

  8. 以下值在将被计算为false,(布尔值与Boolean对象的值是不一样的):

    • false
    • undefined
    • null
    • 0
    • NaN
    • 空字符串 (“”)
  9. 一个闭包必须保存它可见作用域中所有的参数和变量。因为每一次调用传入的参数都可能不同，每一次对外部函数的调用都实际上重新创建了一遍这个闭包。只有当inside的返回值没有再被引用时，内存才会被释放。
  10. 当同一个闭包作用域下两个参数或者变量同名时，就会产生命名冲突。更近的作用域有更高的优先权，所以最近的优先级最高，最远的优先级最低。这就是作用域链。链的第一个元素就是最里面的作用域，最后一个元素便是最外层的作用域。
  11. 闭包中的神奇变量this是非常诡异的。使用它必须十分的小心，因为this指代什么完全取决于函数在何处被调用，而不是在何处被定义。
  12. 函数参数：从ECMAScript 6开始，有两个新的类型的参数：默认参数，剩余参数(rest parameters)。
  13. 箭头函数：
    • 总是匿名的。
    • 有两个因素会影响介绍箭头函数：更简洁的函数和this。
    • 在箭头函数出现之前，每一个新函数都重新定义了自己的this值（在严格模式下，一个新的对象在构造函数里是未定义的，通过上下文对象调用的函数被称为“对象方法”等）。
  14. 预定义函数：
    • eval(),顶级方法，与任何对象无关。
    • isFinite(),全局方法，与任何对象无关。
    • isNaN(),检测是否为 NaN 的值。
    • parseInt(),全局函数,不属于任何对象。
    • parseFloat(),全局函数,不属于任何对象。
    • decodeURI(),
    • decodeURIComponent(),
    • encodeURI(),
    • encodeURIComponent(),
    • uneval(),非标准方法，不要使用。
    • escape(),已废弃。
    • unescape(),已废弃。
  15. 特殊运算符：
    • 条件运算符： condition ? val1 : val2
    • 逗号运算符，分隔作用
    • delete, 删除一个对象或者一个对象的属性, 或者一个数组中某一个键值。
    • in, 如果指定的属性在指定的对象中会返回true。
    • instanceof,
    • new, 与Array, Boolean, Date, Function, Image, Number, Object, Option, RegExp, String 一起使用。
    • this,
    • typeof
    • void
    • super

内置对象

Number

  • Number.prototype有3个方法
    • toExponential(), 返回一个数字的指数形式的字符串。
    • toFixed(), 返回指定小数位数的表示形式。
    • toPrecision(), 返回一个指定有效位数的指数形式的字符串。

Math

  • 不能创建自定义的Math对象，只能使用内置的Math对象。

Date

  • Date 对象的范围是相对距离 UTC1970年1月1日 的前后 100,000,000 天。
  • 处理日期时间的Date对象方法可分为以下几类:
    • “set” 方法, 用于设置Date对象的日期和时间的值。
    • “get” 方法,用于获取Date对象的日期和时间的值。
    • “to” 方法,用于返回Date对象的字符串格式的值。
    • parse 和UTC 方法, 用于解析Date字符串。

String

  • 除非必要, 应该尽量使用String字面值, 因为String对象的某些行为可能并不与直觉一致。
  • 模板字符串：模板字符串使用反勾号 (` `) (grave accent) 包裹内容而不是单引号或双引号. 模板字符串可以包含占位符. 占位符用美元符号和花括号标识 (${expression})。
  • 国际化：Intl 对象是ECMAScript国际化API的命名空间, 它提供了语言敏感的字符串比较，数字格式化和日期时间格式化功能. Collator, NumberFormat, 和 DateTimeFormat 对象的构造函数是Intl对象的属性。
    • DateTimeFormat, 日期和时间格式化方法。
    • NumberFormat, 数字格式化方法。
    • Collator, 定序方法。

RegExp

  • 特殊字符参考表
  • 正则表达式可以被用于RegExp的exec和test方法以及 String的match、replace、search和split方法。

Array

  • JavaScript实际上是将元素作为标准的对象属性来存储，把数组索引作为属性名。
  • forEach() 方法提供乐遍历数组元素的一种方法。
