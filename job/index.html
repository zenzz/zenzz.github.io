项目经验

杭州大搜车汽车服务有限公司项目

车牛

download

该应用是一款以二手车商为目标用户群体，为其提供收车卖车、交流二手车车源、车辆拍卖等服务的产品。

业务层次上大致有以下几个模块：车源显示、订单管理、车牛钱包、账号中心、车牛拍卖、发车业务、车辆管理、即时通讯。项目同时运用了 Native 和 HTML5 技术， Native 和 JavaScript 的交互采用 Jokey + cheniu_protocol + cheniuWebView 框架实现，项目主体框架结构：cheniu_store + cheniu_core + cheniu_protocol + cheniu_auction + 第三方框架。

重庆电信公司项目

爱营销

该产品针对重庆区内线下营业厅销售人员和客户经理以及重庆电信官网销售人员设计。应用经过多次迭代后，业务模块大致有以下几点：客户查询、拍照预受理、楼盘视图、新增客户、CRM 扫码、电子档案、资料返档、订单查询。用户界面 使用代码 + Masonry 布局，身份证信息拍照获取使用 OCR 技术，扫码识别采用 ZBarSDK。

财务辅助/CPMIS

财务辅助和 CPMIS 是两个应用，但是除了网络接口不一样外，项目结构、架构设计、UI布局均是一致的。以HTML 5 作为信息展示层和用户交互层，Native 与 JavaScript 的交互采用第三方框架WebViewJavascriptBridge 实现，Native 处理数据，JavaScript 操作 UI。
